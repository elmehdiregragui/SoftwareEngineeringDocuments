# Clinique Vétérinaire – Site Web

## Description
Ce projet consiste à développer un site web pour une clinique vétérinaire.
Le système permettra de gérer les rendez-vous, les animaux, les propriétaires
et les vétérinaires, avec une organisation claire et évolutive.

## Type d’application
- Application Web
- Projet académique – Software Engineering

## Objectifs
- Gérer les rendez-vous vétérinaires
- Centraliser les informations des animaux et des propriétaires
- Permettre aux vétérinaires de consulter leur planning

## Utilisateurs
- Administrateur
- Vétérinaire
- Réceptionnsite
- Client / Propriétaire
